### Projeto

Este projeto busca implementar o jogo da forca, onde o objetivo é adivinhar uma palavra secreta antes que acabem as tentativas, que diminuem a cada letra adivinhada incorretamente. Utilizamos a linguagem montadora do Processador desenvolvido no Instituto de ciências matemáticas e de computação, portanto, para entender o código por inteiro é necessário conhecer o conjunto de instruções e arquitetura da máquina, que está presente no seguinte [link](https://github.com/simoesusp/Processador-ICMC).

O arquivo "simple_simulator_template" constitui o programa da máquina de controle do processador, onde as instruções são divididas em estado de busca, decodificação e execução e são implementados coordenando registradores. Para a coompreensão total desta implementação, veja a arquitetura do processador no link acima.

### Como executar

Consulte o [seguinte guia](https://github.com/simoesusp/Processador-ICMC/tree/master/Install_Packages) para baixar o .zip adequado ao seu sistema operacional. Após isso:

1. Descompacte o arquivo .zip
2. Abra a pasta "\Simulador\Sublime Text 3"
3. Execute o arquivo "sublime_text"
4. Abra com o sublime text o arquivo "Hello4.asm" na pasta anterior
5. Coloque o código fonte
6. Pressiona F7 para a execução.

Caso o sublime peça para ser atualizado, não o atualize. Dentro do montador, certifique-se de apertar a tecla Home para que ele execute as instruções automaticamente, do contrário eles seriam executadas uma a uma.
